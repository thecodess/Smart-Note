Smart Note is a note app built with Jetpack Compose. You can Add, Delete and Update Notes.
Tech Stack:

 Kotlin
 
 Jetpack Compose
 
 LiveData
 
 ViewModel
 
 Room
 
 Coroutines
 
 Gradle Kotlin DSL
 
 Ktlint
 
 Protobuf
 
 The apk can be found here https://github.com/thecodess/Smart-Note/blob/main/app-debug.apk
